# ts-snake-games

> 项目名称：贪吃蛇游戏 Demo
>
> 项目简介：
>
> 本项目是为学习 TypeScript 所搭建的 Snake Demo
> 可点击查看 [在线 Demo](https://ares-chang.github.io/ts-snake-games/)

## 开发环境

项目通过自建 Webpack 框架，可热重载，兼容低版本浏览器打包等问题。

运行需具备：

- Node 环境
- VSCode 编辑器
- Chrome 浏览器

## 项目须知

项目使用 `yarn` 包管理工具，请统一使用，防止包加载混乱。

## 加载命令

### 下载依赖

```
yarn
```

### 运行项目

```
yarn start
```

### 打包编译

```
yarn build
```
